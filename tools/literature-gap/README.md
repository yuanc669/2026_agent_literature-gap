# literature-gap 工具

文獻研究缺口分析工具。

## 使用方式

對 Claude 說：
- `/literature-gap` — 引導式輸入
- `/literature-gap [論文路徑]` — 分析單篇 PDF
- `/literature-gap [主題關鍵詞]` — 主題式缺口掃描
- `/literature-gap [主題] --cluster [exo|sepsis|uro]` — 指定研究集群
